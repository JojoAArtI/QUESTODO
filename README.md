# QUESTODO
> "The only to-do list that judges you (lovingly)."

**Task Quest** is a simple, TODO app built with **Jetpack Compose** and **Neo-Brutalism** design. It's all you need; 
![App Screenshot](app-screenshot.png)

## Features 

-   **EVERYTHING OFFLINE, FREE, IF YOU WANNA CHANGE SOMETHING JUST CLONE AND ADD IT YOURSELF**
## Tech Stack 

-   **Kotlin** + **Jetpack Compose**
-   **Room Database** (Local Storage)
-   **AlarmManager** (For the pestering)
-   **Hilt** (Dependency Injection - wait, did we use Hilt? No, we kept it simple. Manual DI for the win!)
