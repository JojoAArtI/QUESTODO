# QUESTODO
> "The only to-do list that judges you (lovingly)."

**Task Quest** is a simple, no-nonsense task manager built with **Jetpack Compose** and **Neo-Brutalism** design. It doesn't just store your tasks; it pesters you to actually do them with witty notifications and sassy empty states.
